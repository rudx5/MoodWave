# MoodWave 🌊

MoodWave is a web application that helps users track their mood and improve their mental well-being.

## Features
- Track daily mood
- Simple and clean UI
- Helps users reflect on their emotional patterns

## Built With
- HTML
- CSS
- JavaScript
